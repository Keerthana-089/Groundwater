# Groundwater Contamination Cluster Analysis

## Overview

Groundwater Contamination Cluster Analysis is a machine learning based project that analyzes groundwater quality data to identify contamination patterns and clusters.

The goal of this project is to detect regions with similar contamination characteristics and help understand pollution distribution in groundwater resources.

---

## Features

* Groundwater dataset analysis
* Data preprocessing and cleaning
* Machine learning based clustering
* Identification of contamination patterns
* Testing functionality for model validation

---

## Installation

Clone the repository:

```
git clone https://github.com/Keerthana-089/Groundwater.git
```

Navigate to the project folder:

```
cd groundwater_app
```

Install required dependencies:

```
pip install -r requirements.txt
```

---

## Running the Application

Run the main program:

```
python app.py
```

To run tests:

```
python test_app.py
```
